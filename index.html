About us:

We are Niv Fono and Katya Donchenko, third year students at BGU.

This game website is our second project in a web development course.

We used JQuery in the login and sign in process, when we needed to get the input value from input boxes. In the Settings page we used JQuery to get the user choises of the colors, key board and monsters amount. Also we used it to be able to hide and show the different pages

All these helped us to get the values from the DOM elements in a very easy way.

Through the development process we had some difficulties at the beginning to understand how to work with JQuery syntax and when. Another difficulty was to understand where we need to write the scripts. And the last challenge was to find a smart way to spread the different kinds of balls randomly on the game board.

This project was challenging but that's the best practice we could get in web development!

Good luck

About: Pacman game that allows you to gain score by eating food balls. There are 3 types of balls that are different from each other in their color, size and number of points they give. If you manage to eat all the food - you win. But be careful of the monsters, they are quite smart when you get closer.
